# index.avar.me

Статическая карта проектов [avar.me](https://avar.me). Публикуется через GitHub Pages.

## Локальный просмотр

```bash
python3 -m http.server 8080
```

Откройте http://localhost:8080

## Домен

Файл `CNAME` указывает на `index.avar.me`. В настройках репозитория GitHub → Pages → Custom domain должно совпадать.
